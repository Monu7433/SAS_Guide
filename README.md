## Overview

This repository contains my learning and practice in Clinical SAS Programming and Biostatistics.

It covers SAS programming concepts, statistical procedures, survival analysis, and CDISC standards used in clinical research and clinical trials.

This guide demonstrates my SAS programming skills and understanding of clinical trial data analysis.

---

## Key Topics Covered

### SAS Programming

- Data step programming
- Data import and export
- Data management
- Dataset merging
- Missing value handling
- Macro programming

### Statistical Analysis

- Descriptive statistics (PROC MEANS)
- Frequency analysis (PROC FREQ)
- T-Test (PROC TTEST)
- Regression analysis (PROC REG)
- Logistic regression (PROC LOGISTIC)

### Survival Analysis

- Kaplan–Meier analysis (PROC LIFETEST)
- Cox Proportional Hazards Model (PROC PHREG)
- Hazard Ratio estimation

### Clinical Trial Standards

- CDISC standards
- SDTM basics
- ADaM dataset structure
- ADSL dataset concepts
